soundcheck button system · sunset

A single hierarchy for every button on the site. Rule of thumb: one PRIMARY per view (the gradient), everything else is quieter so the main action always wins. Each button below is shown in every state — and they're all live, so hover / tap them too. Sharp corners, Bricolage, dusk ground, no glow.
